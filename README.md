# Техническое задание: сайт Маши Шишковой

## О проекте

Персональный сайт гида по Амальфитанскому побережью, Неаполю, Помпеям и Капри. Маша Шишкова — гид, преподаватель итальянского языка, организатор квестов, дегустаций и культурных мероприятий. Живёт в Майори (Амальфитанское побережье).

## Целевая аудитория

- Русскоязычные туристы, планирующие поездку в Кампанию (Неаполь, Амальфи, Помпеи, Капри)
- Люди, интересующиеся изучением итальянского языка
- Участники офлайн-мероприятий в Петербурге (лекции, дегустации)

### 2. Экскурсии

Каталог экскурсий с описаниями, ценами и условиями:

- **Неаполь** — [`doc_naples_tours.md`](doc_naples_tours.md)
  - Неаполитанский характер (sacra e profana)
  - Капелла Сан Северо (масоны и легенды)
  - Футбольный Неаполь (от Марадоны до Хвичи)
  - Квест «Голы Марадоны» (для детей)
  - Гастротур
  - Археологический музей
  - Обзорные экскурсии
  - Тень-тур (в жару)

- **Помпеи и Геркуланум** — [`doc_pompeii_herculaneum.md`](doc_pompeii_herculaneum.md)
  - Помпеи для первого раза (3 часа)
  - Глубокое погружение с обедом (5 часов)
  - Геркуланум
  - Виллы (Ариадны, Сан Марко, Поппеи, Реджина)

- **Амальфитанское побережье** — [`doc_amalfi_coast.md`](doc_amalfi_coast.md)
  - Амальфи (блиц и расширенная)
  - Равелло (блиц и полная)
  - Амальфи + Равелло на целый день
  - Мастер-класс по моцарелле
  - Морская прогулка на яхте

- **Многодневный тур** — [`doc_greek_tour_and_more.md`](doc_greek_tour_and_more.md)
  - «Сначала были греки» — 3-дневный археологический экспириенс
  - Дополнительные услуги (консультации, маршруты под ключ)
